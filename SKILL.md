# checkin-skill

## 功能描述

本Skill提供一站式的考勤打卡解决方案，自动执行纷享销客应用的打卡操作，包括：
- 自动检查并启动模拟器
- 启动纷享销客应用
- 智能导航到考勤页面
- 检查打卡状态（定位、时间、按钮状态）
- 执行打卡操作
- 生成打卡结果报告
- 支持集成到Agent中使用

## 环境要求

- Python 3.6 或更高版本
- ADB工具已安装并配置到系统环境变量
- Android模拟器已安装并可正常运行

## 使用方法

### 1. 直接运行

```bash
cd checkin-skill/scripts
python skill_checkin.py
```

### 2. 自动打卡

```bash
cd checkin-skill/scripts
python auto_checkin.py
```

### 3. 集成到Agent中

```python
from scripts.skill_checkin import run_checkin

def user_confirm_callback(status_info):
    """用户确认回调函数"""
    print(f"定位状态: {status_info['location']}")
    print(f"当前时间: {status_info['current_time']}")
    print(f"按钮状态: {status_info['button_status']}")
    # 这里可以实现agent的确认逻辑
    return True  # 自动确认

# 执行打卡操作
result = run_checkin(user_confirm_callback)

# 处理打卡结果
if result['success']:
    print(f"打卡成功！{result['message']}")
else:
    print(f"打卡失败: {result['message']}")
```

## 执行流程

1. **检查模拟器状态**：自动检测模拟器是否运行，未运行则尝试启动
2. **启动纷享销客**：通过ADB命令启动应用
3. **导航到考勤页面**：
   - 企信页面 → 点击应用按钮
   - 应用页面 → 点击考勤选项
   - 考勤页面 → 刷新页面
4. **检查打卡状态**：
   - 检查定位是否在考勤范围内
   - 验证当前时间是否在打卡时段内
   - 检查打卡按钮状态
   - 检测当日打卡记录
5. **用户确认**：展示打卡状态并请求确认（可自动确认）
6. **执行打卡**：根据时间执行对应操作
   - 早上 (08:30-09:00)：执行签到
   - 中午 (12:00-13:00)：执行签退+签到
   - 晚上 (18:00-19:00)：执行签退
7. **汇报结果**：截图记录并返回打卡结果

## 打卡时间规则

| 打卡类型 | 时间范围 | 操作规则 |
|---------|----------|----------|
| 早上上班签到 | 08:30 - 09:00 | 执行签到 1 次 |
| 中午午休打卡 | 12:00 - 13:00 | 执行签退 1 次 + 签到 1 次（无需间隔） |
| 晚上下班签退 | 18:00 - 19:00 | 执行签退 1 次 |

## 注意事项

1. **定位要求**：必须显示「已进入地点考勤范围」才能执行打卡
2. **时间限制**：不在规定时间范围内只汇报情况，不执行补签操作
3. **中午特殊处理**：12:00-13:00会自动执行签退+签到的组合操作
4. **用户确认**：执行打卡前会请求用户确认，可通过回调函数实现自动确认
5. **截图记录**：打卡前后都会自动截图，保存到screenshots目录
6. **模拟器管理**：如模拟器未运行，会自动尝试启动
7. **页面导航**：支持从企信页面和应用页面智能导航到考勤页面

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| ADB连接失败 | ADB未正确安装或模拟器未运行 | 检查ADB安装状态，确保模拟器正常运行 |
| 应用启动失败 | 应用未安装或包名错误 | 确认纷享销客已安装，检查包名配置 |
| 导航失败 | UI布局不同 | 可能需要调整点击坐标 |
| 定位失败 | 模拟器定位服务未开启 | 确保模拟器已开启定位服务 |
| 模拟器启动失败 | Android SDK未安装或模拟器名称错误 | 检查Android SDK安装状态，确认模拟器名称 |
| 打卡按钮点击失败 | 按钮位置变化 | 脚本会自动查找按钮位置，如失败需手动调整 |
